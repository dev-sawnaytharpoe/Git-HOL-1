# Git HOL 1 - Static Web Page

This project is a simple static web page built using HTML and CSS. Its primary purpose is to provide hands-on practice with Git as part of a DevOps assignment.

## Purpose

- Demonstrate basic Git workflows.
- Practice version control with a static website.
- Learn collaboration techniques using Git.

## Getting Started

1. **Clone the repository:**
  ```bash
  git clone <repository-url>
  ```
2. **Navigate to the project directory:**
  ```bash
  cd Git-HOL-1
  ```
3. **Open `index.html` in your browser to view the web page.**
